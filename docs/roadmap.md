# CoachFlow Product Roadmap

## Vision

AI-native, grassroots-first, mobile-first football coaching platform for athletes (primary users) and coaches.

## Current Status

Sprint 0 live (auth + scaffold) at [coachflowapp.netlify.app](https://coachflowapp.netlify.app).

## 13-Sprint Plan

| Sprint | Focus |
|--------|-------|
| 1 (current) | Match Review v1 — athlete logs matches with notes |
| 2 | Coach role + Squad management |
| 3 | Weakness tracking + Player journal (mood/sleep/energy) |
| 4 | AI chatbot + Plan generation + Regenerate-plan UX |
| 5 | YouTube content discovery + Coaching philosophy filter |
| 6 | Chat infrastructure (group + private + AI) with safeguarding |
| 7 | Push notifications + new-message popups |
| 8 | Video upload + telestrator overlay (manual coach annotations) |
| 9 | AI-drawn annotations on uploaded video (Vision API) |
| 10 | Auto-clip + auto-highlight reels |
| 11 | POV camera support + scanning behavior analysis |
| 12 | Parent reports + AI monthly digest emails |
| 13+ | Wearables, smart ball, multi-cam, drone footage |

> **Note:** This order is subject to Sasha's (first user) feedback and may be reshuffled.
