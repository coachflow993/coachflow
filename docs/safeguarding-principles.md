# Safeguarding Principles (Youth Football)

## Overview

CoachFlow handles minors' data. Safeguarding is built in from day one, not bolted on later.

## Adult-Minor Communication Rules

- **NO** direct private chat between adult coach and athlete under 18
- All adult-minor messages must include a parent/guardian as visible participant
- Group chats: all adults can see all messages (transparency)
- Default: parent-mediated only

## Video and Image Consent

- Every athlete (and any visible bystander) needs documented filming consent
- For under-18s, parent/guardian provides consent
- Consent tracker per athlete is mandatory
- Right to withdraw at any time

## POV Camera Ethics (Sprint 11+)

- Training environments only for under-18s, not match days
- All other visible parties must consent to being filmed
- 16+ academy-level athletes only by default
- Hardware compliance with local data protection law

## GDPR/UK Data Protection Requirements

- Data export endpoint per user
- Account/data deletion endpoint
- Clear retention policy (auto-delete inactive accounts after X months)
- Privacy policy plain-language version for parents
- Sub-13 accounts: parent-managed only (COPPA + UK GDPR)

## AI Moderation Requirements

- AI scans group chats for safeguarding flags
- Auto-alert admin coach on flagged content
- Audit log of all flagged content + actions taken
