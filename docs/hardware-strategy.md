# Hardware Strategy (Software-First)

## Philosophy

CoachFlow never builds custom hardware. We integrate with existing consumer hardware from established brands.

## Phase 1 (Sprint 8+): Universal Video Upload

- Accept any video file from any device (phone, GoPro, DJI, drone)
- Normalize formats server-side
- Cap upload size, encourage WiFi-only uploads

## Phase 2 (Year 2): Recommended Hardware Bundles

"CoachFlow-recommended" starter kit:

- Insta360 GO 3 (~£250) for POV
- Magnetic headband mount (~£20)
- Or: cheap action cam clone (~£80) + elastic headband (~£10)

Other recommended hardware:

- GoPro Hero series for full match recording
- Ray-Ban Meta Smart Glasses (16+ users)

## Phase 3 (Year 3+, Contingent on Revenue): Possible Branded Hardware

- Co-branded Insta360 GO partnership OR
- Pure resale of recommended kits via affiliate
- **NOT** custom-built CoachFlow hardware

## Wearables for Future Integration (Sprint 13+)

- GPS vests: Catapult, STATSports, Playermaker, Apex
- Smart insoles/boot sensors: Playermaker, Soccerment
- Smart balls: Adidas/Wilson sensor-equipped
- Consumer wearables: Apple Watch, Garmin, Whoop, Oura

## Hardware Compatibility List (Maintain Over Time)

- GoPro Hero 9, 10, 11, 12, 13
- Insta360 GO 2, GO 3, X3, X4
- DJI Action 3, 4, 5
- Ray-Ban Meta (2nd gen)
- Oakley Meta HSTN (when released)
- Any modern smartphone (iOS 14+, Android 10+)
